# TripleTen
# 📚 Proyecto: Biblioteca Triple Peaks

La página web de la **Biblioteca Triple Peaks** es el segundo proyecto del programa de Desarrollo Web de **TripleTen**. Este sitio fue desarrollado siguiendo un brief de diseño profesional, transformando requerimientos visuales en una interfaz web funcional.

## 🚀 Características del Proyecto
* **HTML5 Semántico:** Estructura de código limpia y accesible.
* **Layout con Flexbox:** Diseño flexible y organizado de elementos.
* **Metodología BEM:** Nombramiento de clases estructurado para un CSS mantenible.
* **Diseño Responsivo:** Adaptabilidad básica para diferentes tamaños de pantalla.

## 🛠️ Tecnologías utilizadas
* HTML5
* CSS3
